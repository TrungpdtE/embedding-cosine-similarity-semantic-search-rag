# Embedding Search Lab

A practical lab for learning embeddings, cosine similarity, semantic search, and the foundations of Retrieval-Augmented Generation (RAG). The repository combines notes, small Python modules, sample data, and optional visualizations.

## What is included

- `docs/`: explanations of embeddings, vectors, similarity, and search.
- `src/`: runnable Python code for embedding and retrieval experiments.
- `data/`: sample text data.
- `notebooks/`: optional exploratory notebooks.
- `embedding_visualization.png`: visual reference for vector concepts.

## Getting started

```bash
python -m venv .venv
pip install -r requirements.txt
```

Run the scripts from `src/` depending on the experiment being studied.

## Learning goals

- Understand what embeddings represent.
- Compare lexical search and semantic search.
- Implement cosine similarity.
- Explain why vector search is a core building block of RAG.
